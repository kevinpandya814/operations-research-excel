# Operations Research & Optimization — Excel Solver

A collection of quantitative optimization projects applying operations research techniques to real-world business problems using Microsoft Excel Solver. Each project demonstrates constraint modelling, sensitivity analysis, and data-driven decision making.

---

## Projects

| # | Project | Technique | Solver Engine | Outcome |
|---|---------|-----------|---------------|---------|
| 1 | [EOQ Inventory Optimization](./MOD_4_EOQ_Inventory/) | Economic Order Quantity | GRG Nonlinear | Minimized annual cost to **$17,060** |
| 2 | [LP Product Mix Optimization](./MOD_5_LP_Product_Mix/) | Linear Programming | Simplex LP | Maximized profit to **$142,051** |
| 3 | [Waste Transportation Optimization](./MOD_6_Transportation_Portfolio/) | Network / Transportation LP | Simplex LP | Reduced shipping cost by **9.2%** |
| 4 | [Portfolio Risk-Return Optimization](./MOD_6_Transportation_Portfolio/#part-2-portfolio-risk-return-optimization) | Quadratic Programming | GRG Nonlinear | Minimized risk at **11% target return** |

---

## Skills Demonstrated

- **Excel Solver** — GRG Nonlinear and Simplex LP engines
- **Linear Programming** — objective function formulation, constraint modelling, feasibility
- **Sensitivity Analysis** — shadow prices, reduced costs, allowable ranges
- **What-If / Data Tables** — two-variable scenario tables for cost surface mapping
- **Network Optimization** — transportation models with transshipment nodes
- **Quadratic Programming** — portfolio variance minimization with a covariance matrix
- **Decision Variable Interpretation** — binding vs. non-binding constraints, slack analysis
